# Spam Email Classifier

## Description
This project is a Spam Email Classifier built using Python and Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Dataset
- spam.csv

## How to Run
1. Install Python
2. Install required libraries:
   pip install pandas scikit-learn
3. Run:
   python main.py
